Text-Based Adventure Game Enhanced by Custom GPT-3 Models

Game AI Final Project, 2023

Language: Python

GitHub

This short text-based adventure game uses custom OpenAI GPT-3 models to allow for a wider breadth of acceptable answers for each prompt.

Some of my contributions to development of this game and system include:
  • Fine tuning custom GPT-3 models to classify a player's response into predefined response types
  • Designing an event system that interfaced with the custom GPT-3 models. Using a player's classified response, the system then selects which outcome out of a predefined set is the most appropriate to play.
