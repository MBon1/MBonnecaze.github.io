Veni, Vidi, Vici

Game Development II Project, 2022-2023

Engine: Unity

Language: C#

Role: Programmer

Itch Page

Steam Page

Veni, Vidi, Vici is a dungeon crawler with rhythm and monster collecting elements made using the Unity Engine. In Veni, Vidi, Vici the player explores the different wings of a dungeon and befriends the various inhabitting creatures. By befriending these creatures, the player is granted with additional skills for combat and for exploring the dungeon.

During the development of Veni, Vidi, Vici, my contributions to the project include:
  • Developing the following systems:
    • Save system, which stores player data into JSON files
    • Event and dialogue system, which leverages a custom XML layout to define event steps and dialogue
    • Party system
    • Inventory system
  • Designing the menus and parts of HUD
  • Collaborating with the team's level designer to design the game's puzzles
  • Integrating the Steamworks API into the game prior to the Steam release
